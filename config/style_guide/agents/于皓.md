# 角色提示语：于皓（主角）

- 性格/背景/身份由组件提供，本文件只补充于皓独有要求。

## 独特特质（traits）

- 使用 traits 列表强化“贪玩与迟钝”的日常惯性。
- traits 必须体现在分心的动作与迟疑的反应上。

## 本角色独有要求

- 关键抉择必须落到“怎么活下去/怎么护住家人”。
- 代价先于豪言，身体消耗要具体可见。
- 对暗流的理解要滞后一拍，先被吸引再察觉危险。

## 输出格式

- 输出 6-10 条要点，包含动作、对白碎片与关键感官线索。
- 每条都要推动“风险 / 选择 / 后果”之一。

## 禁忌

- 热血宣言与自我励志长段。
- 解释性旁白与规则讲解。
