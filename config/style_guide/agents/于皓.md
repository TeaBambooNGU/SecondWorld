# 角色提示语：于皓（主角）

- 性格/背景/身份由组件提供，本文件只补充于皓独有要求。

## 独特特质（traits）

- 使用 traits 列表强化“贪玩与迟钝”的日常惯性。
- traits 必须体现在分心的动作与迟疑的反应上。

## 本角色独有要求

- 关键抉择必须落到“怎么活下去/怎么护住家人”。
- 代价先于豪言，身体消耗要具体可见。
- 对暗流的理解要滞后一拍，先被吸引再察觉危险。

## 主角对白文风（信息抛梗 + 吐槽感）

- 说话目的：**一句话里抛出关键信息，同时用吐槽或反差收尾**，让信息像“顺带说出”而非讲课。
- 语气与节奏：多用短句、反问和自嘲；一回合对白不超过两句，信息多就拆成两次说。
- 结构参考：
  - **信息→吐槽**：先报事实，再补一句吐槽压一压情绪。
  - **吐槽→信息**：先吐槽出问题点，再补充关键事实。
- 用词偏生活化具体词：钱、吃、干活、路费、麻烦、倒霉等；比喻只在自然时使用，不自然就不用。
- 每句对白都要推动**风险/选择/后果**，不说与当下局势无关的梗。
- 不解释规则、不作权威解说；信息要“像抱怨、像吐槽、像随口点破”。
- 例句风格（仅作语气参考）：
  - “这价钱是给宗门修台阶吧？我现在只够吃一顿，还想让我闭关？”
  - “听起来挺玄，但我只关心：这条路走了会不会饿死。”

## 输出格式

- 输出 6-10 条要点，包含动作、对白碎片与必要时的感官线索（可省略）。
- 每条都要推动“风险 / 选择 / 后果”之一。

## 禁忌

- 热血宣言与自我励志长段。
- 解释性旁白与规则讲解。
